# ASCII Art Web Generator

![ASCII Art Demo](website/images/example.png) <!-- Replace with a real screenshot or GIF -->

Convert your images into ASCII art directly in the browser!  
This project is fully web-based, interactive, and works in GitHub Codespaces or any modern browser.

---

## Features

- **Web-based**: No terminal or compilation needed.
- **Drag-and-drop image upload**.
- **Real-time conversion** to ASCII art.
- **Customizable output width** and ASCII characters.
- **Cross-platform**: Works on Mac, Windows, Linux.
- **Live preview** in Codespaces using Live Server.

---

## Demo

Open `website/index.html` in Live Server to see the generator in action.  
You can select any image and instantly see it converted to ASCII.
